## 專案概述

### 問題背景

國際學生在台灣面臨的支付痛點：

| 問題 | 影響 |
|------|------|
| 跨國支付系統分散 | 需要多個 App、多張卡片 |
| 高額手續費 | 跨境轉帳成本高 ||
| 信任與安全疑慮 | 不熟悉當地金融環境 |

**市場規模**：全球每年 690 萬留學生，金融工具卻沒有跟上需求。

---

### 解決方案

TaiPAY 是一個 **Closed-loop 校園支付閘道**，整合學生、學校、商家：

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   學生      │────▶│   TaiPAY    │────▶│   商家      │
│ (Student ID)│     │  (Escrow)   │     │ (校內店家)   │
└─────────────┘     └─────────────┘     └─────────────┘
                           │
                    ┌──────┴──────┐
                    │     PSP     │
                    │ (金流服務商) │
                    └─────────────┘
```

### 核心價值

| 特點 | 說明 |
|------|------|
| **即時** | P2P 轉帳秒到 |
| **零手續費** | 校內轉帳免費 |
| **安全** | 學號驗證 + PSP 託管 |
| **校園整合** | 連結學生證系統 |

## 📊 商業分析

### 市場規模估算 (TAM/SAM/SOM)

| 層級 | 計算方式 | 金額 |
|------|----------|------|
| **TAM** | 109 萬大學生 × NT$255/人 | NT$2.78 億/年 |
| **SAM** | TAM × 90% 智慧型手機滲透率 | NT$2.50 億/年 |
| **SOM** | SAM × 65% 可觸及市場 | NT$1.63 億/年 |

### 商業模式

```
Revenue Streams
├── Freemium 訂閱
│   ├── Free: 10 次/月
│   └── Premium: NT$100/月 無限次
│
├── Float Revenue
│   └── 閒置餘額利息收入（與 PSP 分潤）
│
└── Campus Advertising
    └── 活動推廣 NT$500-1000/檔
```

**預估月營收**：NT$175 萬（10% 付費轉換率）

### 競爭分析

| 功能 | LINE Pay | Wise | Revolut | **TaiPAY** |
|------|:--------:|:----:|:-------:|:----------:|
| 學號登入 | ❌ | ❌ | ❌ | ✅ |
| 為學生設計 | ❌ | ✅ | ✅ | ✅ |
| 校園整合 | ❌ | ❌ | ❌ | ✅ |
| P2P 免手續費 | ✅ | ❌ | ❌ | ✅ |
